# CTF

Il y a trois CTF différents; vous n'êtes pas obligés de les résoudre dans un ordre précis.

Les flags suivent tous le format: `flag-(...)`.

Inscrivez chaque flag dans le formulaire Google prévu à cet effet.

## Challenge #1

Dans ce challenge, vous devez contourner une page d'authentification pour récupérer le flag.

URL du site: `https://ctf1.natbo.ca/`

## Challenge 2

Voici l'url pour le deuxième challenge: `https://ctf2.natbo.ca/`


## Challenge #3
Pour ce challenge vous aller devoir utiliser un décompillateur afin de trouver le dernier flag. 

Vous disposez du code source partiellement complété ainsi que du binaire compilé. Votre objectif: analyser la version compilée avec un décompilateur et en extraire le flag.
