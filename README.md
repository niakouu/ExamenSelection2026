# Bienvenue à l'examen de qualification des CsGames 2026

Cet examen comprend 11 challenges distincts :
- Agents intelligents
- Intelligence artificielle
- Cobol
- Data Management (SQL)
- DevOps (CI/CD)
- High Performance Computing en Rust (bibliothèques CubeCl et Burn)
- Preuve en lean4
- Shaders en OpenGL (GLSL, C++)
- UI-UX (Figma)
- Web (React, Express, TypeScript)

Pour les consignes propres à chaque challenge, consultez le `README.md` du dossier correspondant.

Il n'est ni nécessaire ni attendu de réaliser tous les défis. En compléter un entièrement est déjà très bien.

Respectez les règles et directives présentées en classe pour la tenue de l'examen.

Bonne chance !
