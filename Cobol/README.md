# Cobol

Pour réaliser ce défi, vous devez compléter les programmes Cobol comme décrit ci-dessous.

Pour exécuter un programme Cobol, vous devez d'abord installer un compilateur :
```bash
sudo apt install gnucobol3
```
Ou utiliser un compilateur Cobol en ligne comme [celui-ci](https://onecompiler.com/cobol).

## 1. FizzBuzz

Écrivez un programme en Cobol qui parcourt les nombres de 1 à 100. 
Si un nombre est divisible par 3, affichez "Fizz".
Si un nombre est divisible par 5, affichez "Buzz".
Si un nombre est divisible par 3 et par 5, affichez "FizzBuzz".

Par exemple :
```
1 
2
Fizz
4
Buzz
Fizz
7
...
14
FizzBuzz
16

```
## 2. Calculer le 1980e nombre de la suite de Fibonacci

Écrivez un programme Cobol qui calcule et affiche le 1980e nombre de la [suite de Fibonacci](https://en.wikipedia.org/wiki/Fibonacci_sequence).

Vous pouvez implémenter cette fonction en utilisant une approche itérative et pour des des points bonus, de façon récursive.

Par exemple : \
Le premier nombre est : 1 \
Le deuxième nombre est : 1 \
Le troisième nombre est : 2 \
Le quatrième nombre est : 3 \
Etc...


## 3. Tri à bulles (Bubble sort)

Écrivez un programme Cobol qui implémente l'algorithme du tri à bulles pour trier un tableau d'entiers en ordre croissant.

Vous devez implémenter la fonction `BUBBLE-SORT-PARA` dans le modèle fourni.
